Joker

by Todd Phillips & Scott Silver. 13 April 2018

This story takes place in its own universe. It has no connection to any of the DC films that have come before it. We see it as a classic Warner Bros. movie. Gritty, intimate and oddly funny, the characters live in the real world and the stakes are personal. Although it is never mentioned in the film, this story takes place in the past. Let's call it 1981.

Det. Garrity : Mr. Fleck, sorry to bother you, I'm Detective Garrity, this is my partner Detective Burke. We had a few questions for you, but you weren't home. So we spoke to your mother.

Joker : You did this to her?

Det. Garrity What? No. We just asked her some questions and she started getting hysterical-- hyperventilating, trouble speaking-- then she collapsed. Hit her head pretty hard.

Joker :They told me she had a stroke.

Det. Garrity Sorry to hear that.

JOKER BURSTS OUT LAUGHING, he can't stop it.

Det. Garrity I'm lost. Is something funny?

The page is a script for the movie Joker. For the project was used HTML/ CSS for style / and JavaScript for event design. The unique feature of the page is that it highlits based on the actor's name. In order to do that, there where used couple stepts.

  • All the lines of the actors where defined into an array using the DOM
  • In order to highlight the lines, a for loop was used through the array with an addEventListener.
  • Finally to make sure only one actor at the time if highlites was used another for loop with if condition, between which actor is highlighted and the line.
